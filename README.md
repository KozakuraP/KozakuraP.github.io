# ColorClock
## 概要
背景色がランダムに変化するシンプルな時計アプリです。
## 本バージョン (P.104) について
このバージョンは時計の設定(秒、曜日の表示など)を追加したものです。
## 今後の予定
- タイルの変化に規則性を持たせる機能 
- タイマー、ストップウォッチなどの追加機能  
- その他の機能追加・改善  
次回の更新は2026/8/10頃を予定しています。タイルの変化を完全なランダムではなく、時刻による規則性をもたせたバージョンです。  
今後、予定は変更になる可能性があります。
