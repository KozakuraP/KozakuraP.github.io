# ColorClock
## 概要
背景色がランダムに変化するシンプルな時計アプリです。
## 本バージョン (P.103) について
このバージョンは、背景色が変化する時計のみを実装したシンプルなものです。
## 今後の予定
* 時刻表示フォーマットの変更機能
* タイルの変化に規則性を持たせる機能（時刻と連動するなど）
* タイマー、アラームなどの追加機能
* その他の機能追加・改善
次回の更新は2026/7/10程度を予定しています。時刻のフォーマット設定を追加したバージョンです。
今後、予定は変更になる可能性があります。
